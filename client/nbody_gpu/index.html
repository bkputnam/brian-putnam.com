WebGPU Required

Your browser doesn't support the WebGPU API, which is required to run this simulation.

Support as of Oct 2024:

  • Chrome:
    • Windows: should be enabled by default
    • Mac: should be enabled by default
    • Linux: supposedly works if you enable a few flags, but I haven't been able to get it to work
    • TIP: If you think you should be seeing WebGPU in Chrome and you're still not, chrome://gpu can give you an idea of how (or if) the browser is correctly detecting your GPU hardware.
  • Firefox:
    • Enabled by default in Nightly builds, but not available in other channels (like Release, ESR, or Beta). Firefox support lags Chrome, and doesn't yet cover the full spec. However this post makes it sound like work is actively progressing so hopefully we'll see full support sooner rather than later.
  • Edge, Opera, and other Chromium-based browsers:
    • Same as Chrome
  • Safari:
    • No support
